Game Engines


I wrote a basic 2D game engine in C++ using SDL and Lua.
Now I am working on a 3D Game Engine in C++ using GLFW, GLM, and GLEW. Based on an Entity Component System Structure, it already has support for player movement and collisions based on events. I am working on putting a sping on Asteroids (1979) as a proof of concept for the engine!
