Everything is Pizza!
This is a prototype AR project. The idea of this game is to cover the world around you in Pizza. This gamify's ARKit's plane detection feature, and encourages a user to find good tracking points by feeding the phone more information. This game is designed to be played with many people in the same room, all competing to see who can get the most "slices" covered in the same amount of time.
